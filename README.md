# ⚡ EthosFlight
### AI Safety Operating Layer for Aviation

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.21114760.svg)](https://doi.org/10.5281/zenodo.21114760)

EthosFlight is a real-time AI safety governance platform for aviation operations. It provides continuous monitoring, intelligent alerting, and automated compliance audit trails across all aviation stakeholders — pilots, ATC operators, drone operators, and regulators.

---

## 🛡️ Core Engines

| Engine | Function |
|--------|----------|
| **SENTINEL** | Real-time risk detection and alerting |
| **CLARITY** | AI decision explainability layer |
| **HANDOFF** | Human-AI protocol management |
| **GUARDIAN** | Compliance and audit trail automation |
| **ORACLE** | Predictive safety intelligence |

---

## 🚀 Live Features (v1.0)
- Live FAA/NOAA weather ingestion via Aviation Weather API
- SENTINEL alert engine with IFR/LIFR/wind/visibility detection
- GUARDIAN immutable audit logging to Azure SQL
- REST API (FastAPI) with full OpenAPI documentation
- React dashboard with real-time data display

---

## 🛠️ Tech Stack
- **Backend:** Python, FastAPI, pyodbc
- **Database:** Azure SQL (Microsoft Azure)
- **Frontend:** React.js
- **Data Sources:** FAA SWIM, NOAA Aviation Weather API
- **Cloud:** Microsoft Azure (West US 2)
